## Reflection

Working on this component library was a helpful way for me to better understand how React and TypeScript work together in real projects. Before this lab, I understood props and components on a basic level, but actually building reusable pieces of UI made everything click. I had to think not just about getting something to display on the screen, but about how to make it flexible, clean, and easy for someone else to use.

One of the biggest challenges was learning how to design TypeScript interfaces that made sense. I had to pay attention to which props were required, which ones should be optional, and how to use children properly. At first, I kept running into typing errors, but after reviewing the interfaces and making them more organized, the components became much easier to work with. TypeScript ended up helping me catch mistakes early, which made the components more reliable.
